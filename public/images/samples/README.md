## Replacing with AI-Generated Images

When replacing these placeholder photos with proper AI-generated images, use the prompts below.

### restaurant — Japanese restaurant / washoku
```
A cozy traditional Japanese restaurant interior with warm wooden tables, paper lanterns, and seasonal dishes.
Soft ambient lighting. Photorealistic. Wide angle.
```

### beauty-salon — Hair salon

```
A modern, bright hair salon interior with white walls, styling chairs, and soft pink accents.
Clean and minimal aesthetic. Photorealistic.
```

### clinic — Chiropractic / wellness clinic

### construction — Construction / renovation

## How to Replace

1. Generate an image using the prompt above (e.g. with Adobe Firefly, Midjourney, or DALL-E)
2. Resize to the target dimensions listed in the table
3. Save as JPG at quality 80-85 (aim for under 300KB per file)
4. Replace the corresponding file in this directory
5. Run `npm run build` and verify the image displays correctly
